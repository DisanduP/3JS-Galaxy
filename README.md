# 3D Interactive Orrery 🌌

A mesmerizing 3D solar system simulation built with Three.js and custom GLSL shaders, featuring multiple visual themes, particle trails, and interactive controls for an immersive astronomical experience.

## ✨ Features

- **Realistic Orbital Mechanics**: Planets orbit the central star with accurate particle trails
- **Multiple Themes**: Switch between Inferno, Veridian, and Celestial color schemes
- **Interactive Controls**: 
  - Activate resonance for planetary alignment effects
  - Adjust time acceleration for orbital speed
  - Reset camera view
- **Advanced Rendering**: Post-processing with bloom effects for stunning visuals
- **Responsive Design**: Optimized for desktop and mobile devices
- **Custom Shaders**: Hand-crafted GLSL vertex and fragment shaders for materials

## 🛠 Technologies Used

- **Three.js**: 3D graphics library
- **GLSL Shaders**: Custom vertex and fragment shaders
- **JavaScript ES6 Modules**: Modern JavaScript with import maps
- **Post-Processing**: Bloom, tone mapping, and shadow effects

## 🚀 Getting Started

No build process required! Simply clone the repository and open `index.html` in your web browser.

```bash
git clone https://github.com/BinaryVortex/3JS-Galaxy.git
cd 3JS-Galaxy
# Open index.html in your browser
```

## 🎮 Controls

- **Mouse/Touch Drag**: Orbit the camera around the scene
- **Mouse Wheel**: Zoom in/out
- **Reset View Button**: Return to default camera position
- **Activate Resonance Button**: Trigger alignment effects and energy pulses
- **Time Button**: Cycle through 0.5x, 1x, and 3x orbital speed
- **Theme Button**: Switch between visual themes
